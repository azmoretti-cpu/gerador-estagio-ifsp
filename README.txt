# Gerador de Termo de Compromisso de Estágio — IFSP Sorocaba

Arquivos:
- index.html
- style.css
- script.js

Como usar:
1. Descompacte o arquivo ZIP.
2. Abra o arquivo index.html no navegador.
3. Preencha todos os campos obrigatórios.
4. Clique em "Gerar documento".
5. Clique em "Imprimir / Salvar em PDF".

Validações incluídas:
- CPF: 000.000.000-00
- RG: 00.000.000-0
- CNPJ: 00.000.000/0000-00
- CEP: 00000-000
- Datas: dd/mm/aaaa
- Horários: 00:00
- Campos obrigatórios do termo
- Data final não pode ser anterior à data inicial

Observação:
Este gerador não substitui o cadastramento oficial no SUAP-IFSP.
